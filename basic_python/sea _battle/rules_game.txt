Правила игры:
1. Пользователям нужно зарегестрироваться
2. Пользователи выбирают режим расстановки кораблей. 
Ни в том, ни в другом случае пользователи не видят растановку кораблей:
	2.1 Автоматический
	2.2 С помощью загрузки из файлы
3. Пользователи выбирают режим игры:
	3.1 Игрок1 против Игрока2
	3.2 Игрок против компьютера. Где также есть два режима:
		3.2.1 Легкий (Если компьютер ранит корабль игрока, 
		то он этого будет продолжать до тех пор, пока совсем не убьет корабль)
		3.2.2 Сложный
4. Всегда и везде должны должны быть справка о правилах игры и возможность выйти из игры.
5. Интерфейс должен быть дружелюбным: всегда и везде должны быть поясняющие надписи
6. Игроки ходят последовательно.
7. После завершения игры будет предложено заново начать игру
8. Вывод таблицы результатов каждой сессии после конца игры
